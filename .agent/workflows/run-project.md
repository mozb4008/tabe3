description: كيفية تشغيل نظام محقّب الذاتي (الخلفية والواجهة الأمامية)
---

لتشغيل المشروع بالكامل، يرجى اتباع الخطوات التالية:

### 1. تشغيل الخادم (Backend)
افتح نافذة ترمينال (Terminal) جديدة وقم بتبديل المسار إلى مجلد `wallet-backend` ثم نفذ:
```powershell
npm install
npm start
```
*سيقوم الخادم بالعمل على الرابط `http://127.0.0.1:3001`*

### 2. تشغيل الواجهة الأمامية (Frontend)
افتح نافذة ترمينال أخرى وقم بتبديل المسار إلى مجلد `wallet-frontend` ثم نفذ:
```powershell
npm install
npm run dev
```
*سيظهر لك رابط تشغيل الواجهة (غالباً `http://localhost:5173`)*
